# secure-browser-simulator

## Overview
This project simulates a simple web browser using stacks to manage navigation (back/forward) and includes basic security detection for malicious URLs.

## Features
- Stack-based navigation (LIFO)
- Back and forward functionality
- URL security detection (domains + keywords)
- Security alert system

## Concepts Used
- Data Structures (Stack)
- Cybersecurity Basics (phishing detection)
- Object-Oriented Programming (Python)
