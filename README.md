# 🔐 Secure Browser Simulator

## 📌 Overview
This project simulates a simple web browser using Python. It demonstrates how browsers manage navigation (back/forward) using stacks and includes basic security checks to detect potentially malicious URLs.

## 🚀 Features
- Stack-based navigation (LIFO)
- Back and forward functionality
- URL security detection (domains + keywords)
- Security alert system

## 🧠 Concepts Used
- Data Structures (Stacks)
- Object-Oriented Programming (Python)
- Basic Cybersecurity Concepts (phishing detection)

## 🖥️ How It Works
- Users can enter commands like:
  - `visit <url>`
  - `back`
  - `forward`
  - `history`
  - `alerts`
- The program simulates browser behavior using two stacks:
  - Back stack
  - Forward stack

## ⚠️ Limitations
- No real internet connection (simulation only)
- Uses simple keyword-based detection
- Cannot detect new or advanced threats
